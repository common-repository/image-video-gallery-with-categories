== Description ==

Image & Video Gallery with categories plugin will help the WordPress users to achieve a perfect gallery for their website. The user will able to show gallery images and video according to category using shortcode.

== Installation ==

1. Upload the entire `Image & Video Gallery with categories` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= what is shortcode for gallery without category?

Please refer the below shortcode. Make sure to change the keyword from the shortcode "with" to "without", it will automatically hide the categories

Sample shortcode: [gallery_view_without_category id='n'] Please make sure to replace the 'n' with the actual gallery ID. To ge the gallery ID please refer the gallery list page, we have created "Gallery ID" for each gallery.
